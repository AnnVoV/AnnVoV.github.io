如何拼接两个数组arr1 和 arr2 并且去掉重复内容

参考资源 stackoverflow

1.先拼接两个数组内容 利用array1.concat(array2)

    var arr1 = [1,2,3];
    var arr2 = [3,4,5];

    var arr3 = arr1.concat(arr2);
    console.log(arr3);//[1,2,3,3,4,5]
    //得到的是两个数组的直接拼接组成的新的数组

2.如何处理两个数组中的相同内容部分

    通过使用js 的Array.prototype.filter() 方法.

    首先看下filter的用法 arr.filter(callback[, thisArg])
    其中callback用法:
    Function to test each element of the array.
    Invokedwith arguments (element, index, array). Return true to keep the element, false otherwise.

    中文解释下： callback 是指一个用于验证数组中每个元素的匹配方法，这个callback有三个参数element,index,array. 如果这个正则的返回结果为true表示保留该元素，如果返回为false则删掉该元素。

    那我们怎么判断元素是否重复了呢？利用js 的Array.prototype.indexOf() 方法

Array.prototype.indexOf(searchElem) 返回的是第一次出现的该元素的位置

    如果我们先得到混合的数组中的内容，然后遍历每一个元素的IndexOf与当前的位置是否一样，如果一样保留（说明只出现了一次），如果不一样，删掉，说明该元素是重复的.

    var arr1 = [1,2,3];
    var arr2 = [3,4,5];
    var arr3 = arr1.concat(arr2);// [1,2,3,3,4,5];

    var result_arr = arr3.filter(function(elem,pos){
        return arr3.indexOf(elem) == pos;
    });//[1,2,3,4,5]
